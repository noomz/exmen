<task type="auto">
  <name>Task 2: Create ExmenApp.swift with MenuBarExtra</name>
  <files>Exmen/ExmenApp.swift</files>
  <action>
Create the main app entry point with MenuBarExtra scene:

```swift
import SwiftUI

@main
struct ExmenApp: App {
    var body: some Scene {
        MenuBarExtra("Exmen", systemImage: "terminal.fill") {
            Text("Exmen - Coming Soon")
                .padding()
                .frame(width: 280, height: 100)
        }
        .menuBarExtraStyle(.window)
    }
}
```

Key points from RESEARCH.md:
- Use `.window` style (not `.menu`) for dynamic content later
- Use systemImage for now, can customize icon later
- No WindowGroup - this is menu bar only
- Fixed frame for consistent popover size
  </action>
  <verify>File exists at Exmen/ExmenApp.swift with @main struct</verify>
  <done>ExmenApp.swift created with MenuBarExtra scene using .window style</done>
</task>

```xml
    <string>1.0</string>

    <key>LSUIElement</key>

    <key>LSMinimumSystemVersion</key>
    <string>$(MACOSX_DEPLOYMENT_TARGET)</string>
</dict>
</plist>
```

**Critical:** LSUIElement = true hides dock icon. This is essential for menu bar-only apps per RESEARCH.md.
